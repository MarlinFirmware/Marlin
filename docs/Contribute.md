
# Contributions

## Submitting Changes

- Submit **Bug Fixes** as Pull Requests to the [`bugfix-2.0.x`][Bugfix] branch.

- Follow the **[Coding Standards]** to gain points with the maintainers.

- Please submit your questions and concerns to the **[Issue Queue][Issues]**.
